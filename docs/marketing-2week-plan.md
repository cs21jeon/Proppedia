# Proppedia 2주 홍보 계획

## 개요

- **기간**: 2주 (14일)
- **빈도**: 주 3-4회 (총 7-8개 콘텐츠)
- **채널**: 쓰레드 (메인), 인스타그램, 유튜브 쇼츠
- **콘텐츠 형식**: 휴대폰 직접 시연 영상
- **목표**: 앱 인지도 확보 및 다운로드 유도

---

## 채널별 특성

| 채널 | 특성 | 콘텐츠 방향 |
|------|------|------------|
| 쓰레드 | 텍스트 중심, 실시간 소통 | 짧은 팁 + 영상, 일상적 톤 |
| 인스타그램 | 비주얼 중심, 릴스 알고리즘 | 15-30초 릴스, 세로 영상 |
| 유튜브 쇼츠 | 검색 유입, 긴 수명 | 60초 이내, 자막 필수 |

## 콘텐츠 주제 풀 (14개)

### 핵심 기능 시연 (7개)
1. **도로명 검색** - 주소 입력 → 건축물 정보 확인
2. **지번 검색** - "사당동 272-26" 한 번에 검색
3. **지도 검색** - 지도 탭하면 바로 조회
4. **토지 정보** - 공시지가, 용도지역 확인
5. **건축물 정보** - 층수, 면적, 사용승인일
6. **PDF 저장** - 조회 결과 PDF로 저장/공유
7. **즐겨찾기** - 자주 보는 건물 저장

### 활용 시나리오 (4개)
8. **이사 전 체크** - 건물 연식, 승강기 유무
9. **투자 검토** - 공시지가로 시세 가늠
10. **임장 전 준비** - 미리 건축물 정보 파악
11. **계약 전 확인** - 건축물대장 정보 검증

### 비교/차별화 (3개)
12. **정부24 vs Proppedia** - 3분 → 30초
13. **무료로 이 정보를?** - 유료 서비스와 비교
14. **부동산 앱 필수템** - 왜 설치해야 하는지

---

## 1주차 일정 (4개 콘텐츠)

### 1-1. 월요일 - 앱 소개
**주제**: 런칭 소개 + 핵심 가치
**쓰레드 예시**:
```
건물 정보 조회하려고 정부24 들어가서
로그인하고... 메뉴 찾고... 출력하고...

이제 3초면 됩니다.
Proppedia 부동산백과 📱

[시연 영상: 주소 입력 → 결과 화면]
```
**시연**: 앱 실행 → 주소 검색 → 결과 확인 (15-20초)
**3채널 동시 업로드**: 같은 영상을 쓰레드/인스타/쇼츠에

---

### 1-2. 수요일 - 3가지 검색 방법
**주제**: 도로명/지번/지도 검색
**시연**:
- 도로명 검색 (5초)
- 지번 검색 "사당동 272-26" (5초)
- 지도 탭 검색 (5초)
**포인트**: "어떤 방식이든 OK"

---

### 1-3. 금요일 - 토지/건물 정보
**주제**: 공시지가, 건축물 정보 확인
**시연**: 검색 결과에서 토지 정보 → 건물 정보 스크롤
**포인트**: "공시지가, 용도지역, 층수, 면적 한눈에"

---

### 1-4. 일요일 - 활용 시나리오
**주제**: 이사 전 건물 체크
**시연**: 새 집 주소 검색 → 건물 연식, 승강기 확인
**포인트**: "이사가기 전에 건물 체크하세요"

---

## 2주차 일정 (4개 콘텐츠)

### 2-1. 월요일 - PDF 저장/공유
**주제**: 조회 결과 PDF로 저장
**시연**: 결과 화면 → PDF 버튼 → 저장 → 카톡 공유
**포인트**: "건축물 정보 PDF로 공유"

---

### 2-2. 수요일 - 부동산 투자 활용
**주제**: 임장 전 준비
**시연**: 관심 매물 주소 검색 → 정보 확인
**포인트**: "현장 가기 전에 미리 파악"

---

### 2-3. 금요일 - 즐겨찾기/검색기록
**주제**: 자주 보는 건물 저장
**시연**: 검색 → 즐겨찾기 추가 → 목록 확인
**포인트**: "관심 건물 모아보기"

---

### 2-4. 일요일 - 총정리
**주제**: 무료 앱 강조 + CTA
**시연**: 전체 기능 빠르게 훑기 (30초)
**포인트**: "완전 무료, 지금 다운로드"
**CTA**: 다운로드 링크 강조

---

## 영상 제작 가이드

### 촬영
- **방향**: 세로 (9:16)
- **길이**: 15-30초 (쇼츠는 60초까지 가능)
- **화질**: 1080p 이상
- **손떨림**: 삼각대 또는 거치대 권장

### 편집
- **자막**: 필수 (음소거 시청 대비)
- **BGM**: 저작권 프리 음악 (유튜브 오디오 라이브러리)
- **텍스트**: 핵심 포인트 강조

### 업로드
- **쓰레드**: 영상 + 텍스트 설명
- **인스타**: 릴스로 업로드, 피드에도 공유
- **쇼츠**: 제목에 키워드 포함
