웹훅 트리거 - 구글시트 입력 감지

구글시트에 입력하는 걸 트리거로 합니다.


​

구글시트 트리거는 몇초, 몇분마다..

이런 형태로 동작을 합니다.

이걸 "값이 입력될 때" 동작하게 하는게

목표입니다.

​

그러기 위해서는 웹훅을 지원하는

Apps script를 작성해야 합니다.

​

n8n Assistant와 싸워가며?

코드를 작성했습니다.

웹훅 노드의 url을 아래에 넣습니다.


​

Apps Script에서 트리거를 설정합니다.



​

배포를 통해 동작할 준비를 합니다.


코드 노드 - 데이터 추출

코드노드를 통해 입력된 지번을

"시군구", "법정동", "번", "지"로

구분해서 뽑아냅니다.


HTTP Request 노드 - 법정동코드 추출

가장 시간이 많이 걸린 부분인데,

구글시트에 시군구/법정동코드가 저장된

스프레드시트를 만들어 두고

반환받은 시군구, 법정동에 해당하는

코드를 찾아서 반환하게 하는

HTTP Request 노드입니다.

역시나 구글시트를 동작시키기위해

Apps Script를 작성합니다.


​

여기엔 Apps Script 배포된 URL이 사용됩니다.


​

그러면 입력된 이름에서 코드가 추출됩니다.


 

​
