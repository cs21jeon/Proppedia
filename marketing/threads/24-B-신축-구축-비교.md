# 24. 신축 vs 구축, 앱으로 판단하기

## 쓰레드 텍스트

신축이 좋을까, 구축이 좋을까.

정답은 없지만
판단 기준은 있습니다.

부동산백과에서 두 건물을 검색해 보세요.

구축인데 용적률이 낮다?
→ 재건축 기대 가능

신축인데 용적률이 이미 높다?
→ 추가 개발 여지 적음

사용승인일과 용적률.
이 두 가지면 감이 잡힙니다.

## 영상 가이드
- 구축 아파트 검색 → 사용승인일 + 용적률 확인
- 신축 아파트 검색 → 사용승인일 + 용적률 확인
- 포인트: 두 건물 비교
