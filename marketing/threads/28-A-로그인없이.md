# 28. 로그인 없이 바로 쓸 수 있어요

## 쓰레드 텍스트

부동산백과는
회원가입 안 해도 됩니다.

앱 설치하고 바로 검색 가능합니다.

공동인증서? 필요 없어요.
본인인증? 그것도 없습니다.

다만 즐겨찾기나 검색 기록을
다른 기기에서도 보려면
구글 로그인을 해두면 좋습니다.

기본 조회는 로그인 없이.
편의 기능은 구글 로그인으로.

이렇게 쓰시면 됩니다.
