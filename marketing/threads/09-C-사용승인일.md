# 09. 사용승인일 = 건물 나이

## 쓰레드 텍스트

이 건물 몇 년 됐지?

부동산백과에서 검색하면
"사용승인일"이 바로 나옵니다.

1992년이면 34년 된 건물이에요.

재건축 가능 여부를 판단할 때
가장 먼저 보는 항목입니다.

보통 30년 넘으면
재건축 대상으로 검토되기 시작하거든요.

투자 관점에서도 중요하고
이사할 때도 건물 상태 가늠하기 좋습니다.

#사용승인일 #건물나이 #재건축 #부동산백과 #건축물대장

- 포인트: 사용승인일 위치 강조
