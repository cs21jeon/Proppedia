# 쓰레드 콘텐츠 운영 가이드

## 콘텐츠 방향
- 부동산백과 앱의 실질적 활용법 중심
- 짧고 임팩트 있는 텍스트 + 앱 사용 영상 조합
- 하루 1~2개 꾸준히 업로드

## 콘텐츠 카테고리
- A: 앱 기본 사용법 (검색, 조회, 저장)
- B: 상황별 활용 (이사, 투자, 임장, 계약)
- C: 부동산 꿀팁 + 앱 연계
- D: 비교/대결 (정부24 vs 부동산백과 등)
- E: 중개사/전문가 활용

## 파일 형식
- 파일명: XX-카테고리-주제.md
- 본문: 쓰레드 텍스트 (500자 이내)
- 영상 가이드: 어떤 앱 화면을 촬영할지 안내
- 해시태그 포함

## 발행 순서 제안
01~05: 앱 기본 소개 + 핵심 기능
06~15: 상황별 활용 시나리오
16~25: 부동산 지식 + 앱 연계
26~30: 비교/전문가/심화
