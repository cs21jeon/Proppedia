# 04. PDF로 저장해서 카톡으로 보내기

## 쓰레드 텍스트

건물 정보 조회하고
PDF 버튼 누르면 한 장짜리 리포트가 만들어집니다.

카카오톡으로 바로 공유할 수 있어요.

중개사분들은 고객한테 보내기 좋고
투자 스터디 할 때 자료로 쓰기 좋습니다.

정부24에서 뽑으면 4~5페이지인데
부동산백과는 핵심만 한 장으로 정리됩니다.

#부동산백과 #PDF #건축물대장 #카카오톡 #부동산중개

## 영상 가이드
- 건물 조회 → PDF 버튼 터치 → PDF 생성 화면 → 공유 버튼 → 카톡 공유
- 포인트: 깔끔한 한 장 리포트 + 공유 과정
