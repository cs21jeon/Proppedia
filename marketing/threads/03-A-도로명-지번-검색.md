# 03. 도로명? 지번? 둘 다 됩니다

## 쓰레드 텍스트

"테헤란로 152"
"역삼동 735-12"

도로명이든 지번이든
아는 주소 그대로 검색하면 됩니다.

부동산 계약서에는 지번 주소가 많고
일상에서는 도로명을 쓰잖아요.

부동산백과는 둘 다 지원합니다.

입력하면 자동완성도 뜨니까
끝까지 다 치지 않아도 돼요.

#부동산백과 #주소검색 #도로명 #지번 #부동산앱

## 영상 가이드
- 도로명 주소 입력 → 자동완성 선택 → 결과 확인
- 지번 주소 입력 → 자동완성 선택 → 결과 확인
- 포인트: 두 가지 검색 방식 비교
