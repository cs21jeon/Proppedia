# 16. 공시지가 ≠ 실거래가

## 쓰레드 텍스트

공시지가랑 실거래가는
완전히 다른 가격입니다.

공시지가는 정부가 매년 정하는 기준 가격.
세금, 부담금 계산할 때 씁니다.

실거래가는 실제 거래된 가격이에요.

보통 공시지가는 실거래가의
60~80% 수준이라고 보시면 됩니다.

부동산백과에서 공시지가 확인하고
실거래가 공개시스템에서 시세 비교하면
투자 판단이 한결 쉬워집니다.

#공시지가 #실거래가 #부동산투자 #부동산백과 #부동산공부

## 영상 가이드
- 부동산백과에서 공시지가 확인하는 화면
- 포인트: "이 가격이 실거래가가 아닙니다" 느낌 전달
