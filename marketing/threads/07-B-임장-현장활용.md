# 07. 임장 갈 때 이렇게 씁니다

## 쓰레드 텍스트

임장 가기 전에
후보 건물 주소를 미리 검색해 둡니다.

현장에서는 지도 검색으로 전환합니다.

GPS가 켜져 있으니까
내 위치 주변 건물을 바로 터치할 수 있어요.

관심 가는 건물은
즐겨찾기 + PDF 저장.

집에 돌아와서
PDF 비교하면 깔끔하게 정리됩니다.

이게 임장 루틴이에요.

#임장 #부동산투자 #부동산백과 #현장조사 #건축물대장

## 영상 가이드
- 사전: 주소 검색으로 건물 조회하는 장면
- 현장: 지도 탭에서 GPS로 현재 위치 → 주변 건물 터치
- 정리: PDF 저장 → 여러 PDF 비교하는 장면
