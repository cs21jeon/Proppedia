# 프롭넷(Propnet) 블로그를 시작합니다 - 부동산 기술의 새로운 시작

## 안녕하세요, 프롭넷입니다

부동산 정보, 어렵고 복잡하게만 느껴지셨나요?

정부24에서 건축물대장 한 장 떼려면 공인인증서부터 찾아야 하고, 공시지가 조회하려면 여러 사이트를 돌아다녀야 하고, 현장에서 상담 내용을 정리하려면 밤새 메모를 옮겨 적어야 했습니다.

**프롭넷(Propnet)**은 이런 불편함을 기술로 해결하고자 시작된 부동산 기술 서비스입니다.

"부동산 정보의 문턱을 낮추자"는 생각 하나로, 누구나 쉽게 부동산 정보를 확인하고 업무를 효율화할 수 있는 도구들을 만들고 있습니다.

---

## 프롭넷이 제공하는 3가지 서비스

### 1. proppedia (부동산백과) - 부동산 정보 백과사전

건축물대장, 공시지가, 토지 정보를 **앱 하나로** 바로 확인할 수 있는 부동산 백과사전 앱입니다.

**주요 기능:**
- 도로명 주소, 지번 주소, 지도 검색 3가지 방식으로 건물 정보 조회
- 건축물대장 핵심 정보를 한 화면에서 확인
- 연도별 공시지가 변동 추이 확인
- 카카오맵 기반 지도에서 건물 경계선(폴리곤) 시각화
- 검색 결과 PDF 저장 및 카카오톡 공유
- 즐겨찾기와 검색 기록 클라우드 동기화

현재 **Google Play 스토어에 출시**되어 있으며, 누구나 무료로 사용할 수 있습니다.

👉 [부동산백과 소개 페이지](https://goldenrabbit.biz/proppedia/)

---

### 2. Proptalk (프롭톡) - AI 음성 텍스트 변환 협업 플랫폼

부동산 상담, 현장 미팅, 전화 통화 내용을 **녹음 파일 하나로** 텍스트 변환하고 AI가 요약까지 해주는 협업 도구입니다.

**주요 기능:**
- 통화 녹음 파일 업로드 → OpenAI Whisper로 자동 텍스트 변환(STT)
- Claude AI가 변환된 텍스트를 핵심 요약 (마크다운 형식)
- 팀별 채팅방에서 녹음 파일과 요약 결과 공유
- Google Drive 자동 백업
- 실시간 알림으로 팀원간 빠른 소통

**이런 분들에게 추천합니다:**
- 하루에 고객 전화를 수십 통 받는 부동산 중개사
- 현장 임장 시 음성 메모를 남기는 투자자
- 팀 미팅 내용을 효율적으로 공유하고 싶은 영업팀

현재 Google Play에서 테스트 중이며, 곧 정식 출시 예정입니다.

👉 [프롭톡 소개 페이지](https://goldenrabbit.biz/proptalk/)

---

### 3. Propsheet (프롭시트) - 부동산 데이터 관리 도구

엑셀이나 에어테이블 대신, **부동산 데이터에 최적화된** 웹 기반 스프레드시트입니다.

**주요 기능:**
- 매물 정보를 체계적으로 관리 (67개 항목 지원)
- 다양한 뷰(필터/정렬/컬럼 커스텀)로 원하는 데이터만 보기
- 수익률, 투자금 등 자동 계산 수식 지원
- 워크스페이스 기반 팀 협업 (관리자/편집자/뷰어 권한)
- 데이터 공유 링크 생성 및 CSV 내보내기
- 매일 새벽 4시 최신 매물 데이터 자동 동기화

👉 [프롭시트 바로가기](https://goldenrabbit.biz/propsheet/)

## 이 블로그에서 다룰 내용

프롭넷 블로그는 **부동산 정보와 기술**을 주제로 꾸준히 업데이트됩니다.

### 부동산 상식
건축물대장 보는 법, 공시지가 조회, 용도지역 이해하기 등 부동산 거래 전 반드시 알아야 할 기본 지식을 쉽게 풀어드립니다.

### 서비스 활용 가이드
proppedia, Proptalk, Propsheet 각 서비스의 사용법을 스크린샷과 함께 단계별로 안내합니다.

### 부동산 데이터 분석
공시지가 변동 추이, 지역별 건축물 현황 등 데이터 기반의 부동산 인사이트를 공유합니다.

### PropTech 개발 이야기
1인 개발자로서 부동산 앱을 만들어가는 과정, 기술적 도전과 해결 경험을 솔직하게 나눕니다.
